# 🏗️ QuantiConstruct - Calcul d'Aménagement Intérieur

## 📌 Description
QuantiConstruct est une application web permettant de :
- Calculer la surface
- Estimer le coût des matériaux
- Calculer le coût des équipements
- Obtenir le total général

Le projet permet aussi d’enregistrer les résultats via une API.

---

## ⚙️ Fonctionnalités

- ✅ Calcul cuisine
- ✅ Calcul placard
- ✅ Calcul sanitaire
- ✅ Calcul du total
- ✅ Sauvegarde des résultats
- ✅ Affichage des résultats enregistrés

---

## 🧱 Technologies utilisées

- HTML
- CSS
- JavaScript
- Node.js (backend)
- API REST
